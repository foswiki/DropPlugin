---+!! Drop Plugin

Extends TopicInteractionPlugin to add placeholders to a topic that you can drop attachments onto.

%TOC{title="On this page:"}%

---++ Features
Supports adding multiple sites on a page where an attachment can be drag-dropped. The attachment will automatically renamed on upload, and can be re-uploaded at any point.

Drop sites are defined using a Foswiki template. The standard template inserts a link to the attachment in the topic.

#Installation
---++ Installation Instructions
This plugin depends on the !TopicInteractionPlugin for JS components and REST handler, and on !RenderPlugin for a REST handler.
